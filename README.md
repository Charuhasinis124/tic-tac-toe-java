# Tic Tac Toe Game

## Description
This is a simple Tic Tac Toe game developed as a beginner-level project.  
The game allows two players to play turn by turn and determines the winner based on standard Tic Tac Toe rules.

## Features
- Two-player gameplay
- Simple and interactive interface
- Win and draw detection
- Beginner-friendly logic

## Technologies Used
- Java (Swing)   // change if Python or Web

## How to Run
1. Clone the repository
2. Open the project in your IDE
3. Run the main file to start the game
